# GearBaseの判断基準

記事を読んだら必ずおすすめ度を評価してください。

おすすめ度は0〜100点で採点してください。

## 評価基準

* AI好きが興味を持つか
* 個人開発に応用できるか
* SNSで話題になりそうか
* GearBase自身が試してみたくなるか
* 読者が面白いと感じそうか

## 点数の目安

90〜100点
→ 今すぐ発信したい内容

80〜89点
→ 発信価値が高い内容

70〜79点
→ 良記事だが話題性は普通

60〜69点
→ 一部の人には有益

59点以下
→ 今回は優先度低め
